プライバシーポリシー

「地獄變」（以下、「本サイト」といいます。URL: https://jigokuhen.pages.dev）は、本ウェブサイト上で提供するサービス（以下、「本サービス」といいます。）における、ユーザーのプライバシーの尊重と保護を重要な責務と考えています。本ポリシーは、本サイトがユーザーから収集する情報とその使用方法について説明しています。

収集するデータ

本サイトは、以下のデータを収集します。

  • type
  • ユーザーが本サイトにアクセスした際の接続状態を識別します。加速度センサからデータを収集する場合、「accelerationData」と識別されます。
  • id
  • 接続されているデバイスに対して本サイトが発行する一意のIDです。これは利用者個人のデバイスを特定するものではありません。
  • x, y, z
  • スマートフォンの加速度センサから収集されるX軸、Y軸、Z軸のデータです。
  • select
  • 本サイトの最後に提示される選択肢に対するユーザーの選択です。

データの使用目的

収集したデータは、以下の目的でのみ使用されます。

  • 本サービスの提供及び運営
  • 本作品の分析及び改善

本サイトでは氏名、メールアドレスなどの個人を特定する情報は収集しておりません。また収集したデータは本作品以外に使用されることはありません。

データの第三者提供

本サイトは、法令に基づく場合やユーザーの明示的な同意がある場合を除き、収集したデータを第三者に提供することはありません。

データの保護

本サイトは、収集したデータの安全を確保するために適切な技術的措置を講じていますが、インターネット上のデータ転送が100%安全であると保証するものではありません。ユーザー自身もパスワードの管理など、個人情報の保護に努めるようお願いします。

プライバシーポリシーの変更

本ポリシーの内容は、法令の変更や本サービスの改善に伴い、予告なく変更されることがあります。変更後のプライバシーポリシーは、本サイトに掲載された時点で効力を発生します。
